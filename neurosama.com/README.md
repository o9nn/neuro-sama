# Neuro-Sama Identity Framework & Developer Resources

This directory contains enhanced web resources for the Neuro-Sama Identity Framework.

## Files

- **index.html** - Main neurosama.com landing page with playful design
- **developer.html** - Developer portal with comprehensive framework documentation
- **style.css** - Styles for the main landing page
- **main.js** - JavaScript for interactive elements
- **color.js** - Color manipulation utilities

## Developer Portal

The developer portal (`developer.html`) provides:

- **Architecture Overview** - Visual representation of the 5-layer architecture
- **Core Components** - Detailed information about Personality Engine, Memory System, and Cognitive Pipeline
- **Quick Start Guide** - Installation and basic usage examples
- **Game Integration Patterns** - Templates for different game types
- **Key Features** - Comprehensive list of framework capabilities
- **Resources** - Links to documentation, SDKs, and examples

## Navigation

From the main page (index.html), click the "Developer Portal - Identity Framework" button to access the developer resources.

## Architecture Highlights

The Identity Framework implements a 5-layer cognitive architecture:

1. **Communication Protocol** - WebSocket-based messaging
2. **Cognitive Integration** - Personality, Memory, Cognition
3. **Domain Adapters** - Game-specific handlers
4. **Rendering Pipeline** - Visual and audio output
5. **Identity Projection** - Public personality and brand
